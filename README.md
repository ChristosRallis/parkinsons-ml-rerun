# Ανίχνευση της Νόσου Parkinson από Φωνητικά Χαρακτηριστικά με Χρήση Μηχανικής Μάθησης
## Εισαγωγή και επισκόπηση

Το παρόν έργο διερευνά τη δυνατότητα διάκρισης ατόμων με νόσο Parkinson από υγιή άτομα ελέγχου, χρησιμοποιώντας ακουστικά χαρακτηριστικά της φωνής και μεθόδους μηχανικής μάθησης. Η προσέγγιση βασίζεται σε ένα σύνολο φωνητικών μετρήσεων, όπως χαρακτηριστικά θεμελιώδους συχνότητας, jitter, shimmer και μη γραμμικά χαρακτηριστικά της φωνής.

Βασικός στόχος του έργου δεν είναι μόνο η επίτευξη υψηλής ταξινομικής απόδοσης, αλλά η συστηματική σύγκριση διαφορετικών συνδυασμών επιλογής χαρακτηριστικών και αλγορίθμων ταξινόμησης. Συγκεκριμένα, αξιολογούνται τέσσερις στρατηγικές: χρήση όλων των διαθέσιμων χαρακτηριστικών χωρίς επιλογή χαρακτηριστικών, επιλογή με ANOVA F-test, επιλογή με LASSO και επιλογή χαρακτηριστικών με ExtraTrees.

Για κάθε στρατηγική αξιολογούνται οι αλγόριθμοι Logistic Regression, Linear Support Vector Machine, Random Forest, XGBoost και Multi-Layer Perceptron. Η πειραματική διαδικασία περιλαμβάνει subject-aware διαχωρισμό εκπαίδευσης και εξωτερικού ελέγχου, βελτιστοποίηση υπερπαραμέτρων, αξιολόγηση με cross-validation, προσαρμογή του decision threshold με έμφαση στην υψηλή ανάκληση και τελική αξιολόγηση σε ανεξάρτητο test set.

Με αυτόν τον τρόπο, το έργο επιδιώκει να εντοπίσει ποιοι συνδυασμοί επιλογής χαρακτηριστικών και ταξινομητών αποδίδουν καλύτερα στο συγκεκριμένο πρόβλημα, καθώς και ποια φωνητικά χαρακτηριστικά εμφανίζονται σταθερά ως χρήσιμα σε διαφορετικές πειραματικές διαμορφώσεις. Τα αποτελέσματα αφορούν το συγκεκριμένο dataset και δεν αποτελούν κλινικό διαγνωστικό εργαλείο.

## Κύριες συνεισφορές

* Υλοποίηση ενός αναπαραγώγιμου πειραματικού πλαισίου για τη διάκριση ατόμων με νόσο Parkinson και υγιών ατόμων ελέγχου από 22 φωνητικά χαρακτηριστικά.

* Συστηματική αξιολόγηση 20 συνδυασμών, που προκύπτουν από 4 στρατηγικές χαρακτηριστικών και 5 αλγορίθμους ταξινόμησης.

* Σύγκριση τεσσάρων προσεγγίσεων: χρήση όλων των χαρακτηριστικών χωρίς επιλογή χαρακτηριστικών (*No Feature Selection*), επιλογή χαρακτηριστικών με *ANOVA F-test*, LASSO και ExtraTrees.

* Αξιολόγηση των αλγορίθμων Logistic Regression, Linear SVM, Random Forest, XGBoost και Multi-Layer Perceptron.

* Χρήση subject-level διαχωρισμού, ώστε εγγραφές του ίδιου ατόμου να μην εμφανίζονται ταυτόχρονα στα δεδομένα εκπαίδευσης και τελικού ελέγχου.

* Συνδυασμός βελτιστοποίησης υπερπαραμέτρων, cross-validation, προσαρμογής του threshold απόφασης και τελικής αξιολόγησης σε ανεξάρτητο test set.

* Δημιουργία πινάκων επιλογής χαρακτηριστικών, ώστε να εξετάζεται ποια φωνητικά χαρακτηριστικά εμφανίζονται συχνότερα ως χρήσιμα στις διαφορετικές πειραματικές διαμορφώσεις.

* Κατάταξη των μοντέλων με βάση την τελική απόδοση, το ROC-AUC, τον αριθμό των χρησιμοποιούμενων χαρακτηριστικών και τη σταθερότητα της απόδοσής τους.

## Σύνολο δεδομένων

Το έργο χρησιμοποιεί το **Parkinson’s Disease Data Set (Oxford Parkinson’s Disease Dataset)** από το UCI Machine Learning Repository. Πρόκειται για πρόβλημα δυαδικής ταξινόμησης, όπου στόχος είναι η διάκριση μεταξύ δειγμάτων ατόμων με νόσο Parkinson και υγιών ατόμων ελέγχου.

Το dataset περιλαμβάνει 195 καταγραφές παρατεταμένης φώνησης από 32 διαφορετικά άτομα. Από αυτές, 147 καταγραφές ανήκουν στην κλάση Parkinson και 48 στην κλάση Healthy. Κάθε άτομο διαθέτει πολλαπλές ηχογραφήσεις, επομένως τα δείγματα δεν μπορούν να θεωρηθούν πλήρως ανεξάρτητα μεταξύ τους σε επίπεδο ατόμου.

Κάθε εγγραφή περιγράφεται από 22 ακουστικά χαρακτηριστικά της φωνής. Αυτά περιλαμβάνουν μετρήσεις θεμελιώδους συχνότητας, jitter, shimmer, λόγους θορύβου και αρμονικότητας, καθώς και μη γραμμικά χαρακτηριστικά δυναμικής φωνής.

| Κατηγορία                  | Παραδείγματα χαρακτηριστικών                     |
| -------------------------- | ------------------------------------------------ |
| Θεμελιώδης συχνότητα       | `MDVP:Fo(Hz)`, `MDVP:Fhi(Hz)`, `MDVP:Flo(Hz)`    |
| Μεταβλητότητα συχνότητας   | `MDVP:Jitter(%)`, `MDVP:RAP`, `MDVP:PPQ`         |
| Μεταβλητότητα έντασης      | `MDVP:Shimmer`, `MDVP:APQ`, `Shimmer:DDA`        |
| Θόρυβος και αρμονικότητα   | `NHR`, `HNR`                                     |
| Μη γραμμικά χαρακτηριστικά | `RPDE`, `DFA`, `spread1`, `spread2`, `D2`, `PPE` |

Η στήλη `name` χρησιμοποιείται μόνο για την αναγνώριση του ατόμου από το οποίο προέρχεται κάθε καταγραφή. Δεν χρησιμοποιείται ως χαρακτηριστικό εισόδου στο μοντέλο. Η μεταβλητή `status` αποτελεί την ετικέτα στόχο, όπου `1` αντιστοιχεί σε Parkinson και `0` σε Healthy.

## Πειραματική μεθοδολογία

Η μεθοδολογία σχεδιάστηκε ώστε να αποφεύγεται η διαρροή πληροφορίας μεταξύ δεδομένων του ίδιου ατόμου και να επιτρέπεται δίκαιη σύγκριση όλων των μοντέλων.

### Διαχωρισμός δεδομένων σε επίπεδο ατόμου

Αρχικά, τα δεδομένα χωρίστηκαν σε external training set και external test set σε επίπεδο ατόμου και όχι απλώς σε επίπεδο εγγραφών. Το 20% των ατόμων χρησιμοποιήθηκε ως ανεξάρτητο test set.

Έτσι προέκυψαν:

* 25 άτομα και 152 δείγματα στο external training set.
* 7 άτομα και 43 δείγματα στο external test set.

Όλες οι καταγραφές ενός ατόμου ανήκουν αποκλειστικά είτε στο training είτε στο test set. Με αυτόν τον τρόπο, το μοντέλο δεν μπορεί να «αναγνωρίσει» έναν ομιλητή που έχει ήδη δει κατά την εκπαίδευση.

### Πειραματικά pipelines

Αξιολογήθηκαν τέσσερις διαφορετικές στρατηγικές διαχείρισης χαρακτηριστικών:

| Pipeline             | Περιγραφή                                                                      |
| -------------------- | ------------------------------------------------------------------------------ |
| No Feature Selection | Χρήση και των 22 διαθέσιμων χαρακτηριστικών                                    |
| ANOVA F-test         | Επιλογή των top-*k* χαρακτηριστικών μέσω `SelectKBest` και `f_classif`         |
| LASSO                | Επιλογή χαρακτηριστικών μέσω Logistic Regression με L1 regularization          |
| ExtraTrees           | Επιλογή των σημαντικότερων χαρακτηριστικών βάσει tree-based feature importance |

Κάθε pipeline συνδυάστηκε με πέντε ταξινομητές:

* Logistic Regression
* Linear Support Vector Machine
* Random Forest
* XGBoost
* Multi-Layer Perceptron

Συνολικά αξιολογήθηκαν 20 συνδυασμοί pipeline και classifier.

### Cross-validation και βελτιστοποίηση

Στο external training set εφαρμόστηκε 3-fold cross-validation σε επίπεδο ατόμου, με χρήση `StratifiedGroupKFold`. Τα ίδια προκαθορισμένα folds χρησιμοποιήθηκαν σε όλα τα pipelines και τους ταξινομητές, ώστε οι συγκρίσεις να γίνονται κάτω από ακριβώς τις ίδιες συνθήκες.

Οι υπερπαράμετροι των classifiers βελτιστοποιήθηκαν αρχικά στο No Feature Selection pipeline με κριτήριο το F1-score. Στα pipelines με επιλογή χαρακτηριστικών, οι βέλτιστες ρυθμίσεις των classifiers διατηρήθηκαν σταθερές και βελτιστοποιήθηκε μόνο η παράμετρος της επιλογής χαρακτηριστικών, όπως ο αριθμός των επιλεγμένων χαρακτηριστικών *k* ή η παράμετρος regularization του LASSO.

Τέλος, για κάθε μοντέλο επιλέχθηκε ξεχωριστό decision threshold από τις out-of-fold προβλέψεις του training set. Η επιλογή έδινε προτεραιότητα σε Recall τουλάχιστον 0.90 και, μεταξύ των κατάλληλων thresholds, μεγιστοποιούσε το F1-score. Η τελική αξιολόγηση πραγματοποιήθηκε μία μόνο φορά στο ανεξάρτητο external test set.

## Αποτελέσματα και βασικά ευρήματα

Η τελική αξιολόγηση πραγματοποιήθηκε στο ανεξάρτητο external test set, το οποίο δεν χρησιμοποιήθηκε κατά την εκπαίδευση, τη βελτιστοποίηση υπερπαραμέτρων ή την επιλογή του decision threshold. Τα πλήρη αποτελέσματα όλων των διαμορφώσεων είναι διαθέσιμα στον φάκελο [`results/model_ranking`](results/model_ranking).

### Σύνοψη αποτελεσμάτων στο external test set

Οι υψηλότερες τιμές F1-score στο συγκεκριμένο external test set καταγράφηκαν από τους συνδυασμούς LASSO feature selection με tree-based classifiers:

| Pipeline             | Classifier             | F1-score | Recall |
| -------------------- | ---------------------- | -------: | -----: |
| LASSO                | Random Forest          |    0.939 |  1.000 |
| LASSO                | XGBoost                |    0.939 |  1.000 |
| No Feature Selection | Multi-Layer Perceptron |    0.925 |  1.000 |
| No Feature Selection | Logistic Regression    |    0.912 |  1.000 |

Τα αποτελέσματα δείχνουν ότι η επιλογή χαρακτηριστικών με LASSO μπορεί, στο συγκεκριμένο dataset και στο συγκεκριμένο πειραματικό πρωτόκολλο, να προσφέρει αποτελεσματικότερη και πιο συμπαγή αναπαράσταση των δεδομένων για ορισμένους ταξινομητές.

Παράλληλα, οι γραμμικοί ταξινομητές παρέμειναν ανταγωνιστικοί σε αρκετές διαμορφώσεις. Αυτό υποδηλώνει ότι τα διαθέσιμα φωνητικά χαρακτηριστικά περιέχουν πληροφορία που μπορεί να αξιοποιηθεί και από σχετικά απλά γραμμικά μοντέλα, χωρίς όμως αυτό να σημαίνει ότι το πρόβλημα είναι πλήρως γραμμικά διαχωρίσιμο.

### Ευρήματα επιλογής χαρακτηριστικών

Για κάθε pipeline επιλογής χαρακτηριστικών δημιουργήθηκαν πίνακες που καταγράφουν ποια φωνητικά χαρακτηριστικά επιλέχθηκαν από κάθε ταξινομητή. Οι πίνακες αυτοί επιτρέπουν την εξέταση της συμφωνίας μεταξύ διαφορετικών μεθόδων και μοντέλων.

Το No Feature Selection pipeline δεν χρησιμοποιείται ως ένδειξη σημαντικότητας χαρακτηριστικών, καθώς σε αυτό διατηρούνται εξ ορισμού και τα 22 διαθέσιμα χαρακτηριστικά. Αντίθετα, λειτουργεί ως baseline για να συγκριθεί αν η επιλογή μικρότερου συνόλου χαρακτηριστικών προσφέρει όφελος.

Ενδεικτικά, στο pipeline ExtraTrees, τα χαρακτηριστικά `MDVP:Fo(Hz)`, `MDVP:Flo(Hz)`, `spread1`, `D2` και `PPE` επιλέχθηκαν και από τις τέσσερις κλασικές διαμορφώσεις ταξινομητών: Logistic Regression, Linear SVM, Random Forest και XGBoost.

Η επαναλαμβανόμενη επιλογή ενός χαρακτηριστικού από διαφορετικές διαμορφώσεις αποτελεί ένδειξη χρησιμότητας μέσα στο συγκεκριμένο dataset και experimental setup. Δεν αποτελεί, όμως, απόδειξη αιτιώδους σχέσης ή κλινικής διαγνωστικής αξίας από μόνη της.

## Περιορισμοί

Τα αποτελέσματα του έργου πρέπει να ερμηνεύονται μέσα στο πλαίσιο του συγκεκριμένου dataset και της συγκεκριμένης πειραματικής διαδικασίας.

* Το dataset είναι σχετικά μικρό και περιλαμβάνει πολλαπλές καταγραφές ανά άτομο. Παρότι χρησιμοποιήθηκε διαχωρισμός σε επίπεδο ατόμου για τον περιορισμό της διαρροής πληροφορίας, το ανεξάρτητο external test set περιλαμβάνει περιορισμένο αριθμό ατόμων.

* Η αξιολόγηση πραγματοποιήθηκε σε ένα μόνο δημόσιο dataset. Επομένως, δεν μπορεί να θεωρηθεί ότι οι επιδόσεις θα γενικεύονται απαραίτητα σε διαφορετικούς πληθυσμούς, διαφορετικές συνθήκες ηχογράφησης ή διαφορετικά σύνολα φωνητικών χαρακτηριστικών.

* Η προσαρμογή του decision threshold έδωσε προτεραιότητα σε υψηλό Recall. Αυτή η επιλογή είναι κατάλληλη όταν είναι σημαντικό να μειώνονται τα πιθανά false negatives, αλλά μπορεί να οδηγήσει σε περισσότερα false positives σε σχέση με ένα μοντέλο που χρησιμοποιεί το προεπιλεγμένο threshold.

* Η συχνή επιλογή ενός χαρακτηριστικού από διαφορετικές μεθόδους αποτελεί ένδειξη χρησιμότητας στο συγκεκριμένο πειραματικό πλαίσιο. Δεν αποτελεί απόδειξη αιτιότητας ούτε επιβεβαιώνει από μόνη της κλινική διαγνωστική αξία.

* Το έργο έχει ερευνητικό και εκπαιδευτικό χαρακτήρα. Δεν αποτελεί κλινικό διαγνωστικό εργαλείο και δεν πρέπει να χρησιμοποιείται για ιατρική διάγνωση ή λήψη κλινικών αποφάσεων.
